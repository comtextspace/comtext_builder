# OPDS каталог

[OPDS (Open Publication Distribution System)](https://ru.wikipedia.org/wiki/OPDS) — это стандарт для распространения цифровых публикаций, основанный на Atom и HTTP.

## Что такое OPDS?

OPDS позволяет электронным библиотекам и читалкам автоматически получать информацию о доступных книгах, их метаданных и ссылках для скачивания. Это открытый стандарт, который упрощает распространение и чтение электронных книг.

## Каталог на этом сайте

На этом сайте доступен OPDS-каталог со всеми книгами:

**[Открыть каталог OPDS →](/catalog.opds)**

Каталог содержит информацию о всех книгах на сайте, включая:
- Название книги
- Авторов
- Ссылки для скачивания в формате FB2

Вы можете использовать этот каталог с любым OPDS-совместимым приложением для чтения электронных книг.

## Как использовать?

1. Скопируйте ссылку на каталог: <CatalogLink />
2. Добавьте её в ваше приложение для чтения книг (например, FBReader, Cool Reader и другие)
3. Приложение автоматически загрузит список доступных книг
4. Вы сможете скачивать и читать книги прямо из приложения
