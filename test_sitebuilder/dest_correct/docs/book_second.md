# Вторая книга

Текст книги

![](/img/book_second-01.png)

> **Стихотворение**
>
> Дева юга! Пред тобою \
> Бездыханен я стою: \
> Взором адским, как стрелою, \
> Ты пронзила грудь мою!
>
> Этим взором, этим взглядом — \
> Чаровница — ты мне вновь \
> Азиятским злейшим ядом \
> Отравила в сердце кровь!

## Формулы

Формула внутри строки: $E = mc^2$ — это знаменитое уравнение Эйнштейна.

Формула на отдельной строке:

$$
1 + \frac{1}{2}
$$

## Примеры использования кода в тексте

`field` — это кодовое слово, которое используется в начале абзаца для обозначения важного термина.

Этот участок текста заканчивается кодовым словом `mycode`, что позволяет завершить мысль техническим термином.

В середине абзаца можно использовать `command` для выполнения определенных действий, что делает текст более информативным.

Комбинированный пример: используйте `git commit` для сохранения изменений, а затем `git push` для отправки в репозиторий. Это стандартный рабочий процесс.

Пример с форматированием: **важно** использовать `sudo` с осторожностью, так как это может привести к *необратимым* изменениям в системе.

Многострочный код в абзаце: сначала выполните `npm install`, затем `npm run build`, и наконец `npm start` для запуска приложения.

Код на уровне абзаца:

```
function example() {
    console.log("Hello, world!");
    return true;
}
```
